== Welcome to the Universal Eyeglass Socket (UES) project

This project aims to create an industry-wide standard for interchangeable prescription eyeglass lenses, enabling consumers to use a single set of prescription lenses across multiple compatible frames.

== Project Documentation

This documentation is organized into three main sections:

<<spec/technical-spec.adoc#,Technical Specification>>::
The complete technical specification defining the physical, optical, and mechanical standards for standardized lens modules.

<<docs/ues-if.adoc#,UES Implementation forum>>::
The concept, market analysis, and business process surrounding the eyeglass socket initiative.

<<docs/plan.adoc#,Plan>>::
Action plan

<<docs/sizes-rationale.adoc#,Frame Size Optimisation>>::
Statistical analysis deriving optimal standardised frame sizes for the EU27 population.
